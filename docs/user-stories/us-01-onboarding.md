# US-01：安装向导 & 配置向导

**模块**：Onboarding
**对应设计**：`design/ui-onboarding.html`、`docs/prd-v2.1.md`

---

### US-01-01：零终端一键安装 OpenClaw

**作为** 非技术用户，
**我希望** 在 lysmata 应用内点击一个按钮即可完成 OpenClaw 的全部安装，
**以便** 不需要打开终端或手动执行任何命令。

**优先级**：P0
**验收标准（AC）**：
- AC1：点击"开始安装"后，界面自动执行官方安装脚本，全程无需用户输入
- AC2：安装过程中显示实时进度条和终端日志滚动区
- AC3：安装成功后自动跳转至配置向导，并发送桌面通知
- AC4：安装失败时给出明确错误信息和重试按钮

---

### US-01-02：环境预检与状态反馈

**作为** AI 开发者，
**我希望** 在安装前看到系统环境（Node.js、权限、网络）的检测结果，
**以便** 提前发现问题，避免安装中途失败。

**优先级**：P0
**验收标准（AC）**：
- AC1：启动安装向导时自动扫描 Node.js 版本、系统权限、网络连通性
- AC2：每项检测以绿/黄/红状态灯直观呈现结果
- AC3：发现问题时给出具体原因和修复建议
- AC4：所有项目通过后"一键安装"按钮才可点击

---

### US-01-03：已安装用户跳过安装直接进入配置

**作为** 已安装过 OpenClaw 的用户，
**我希望** 在欢迎页选择"已安装，直接配置"，
**以便** 跳过安装步骤，节省时间。

**优先级**：P0
**验收标准（AC）**：
- AC1：Intro 页显示"已安装 OpenClaw，直接配置 →"入口
- AC2：点击后直接跳转到配置向导 step1（Gateway 配置）
- AC3：跳过安装步骤不影响后续配置流程

---

### US-01-04：可视化 Gateway 配置

**作为** AI 开发者，
**我希望** 通过表单界面配置 Gateway 端口、绑定地址、认证模式和开机自启，
**以便** 不需要手动编辑 JSON 配置文件。

**优先级**：P0
**验收标准（AC）**：
- AC1：表单提供端口（默认 18789）、绑定地址（默认 127.0.0.1）、认证模式下拉选择
- AC2：开机自启通过 Toggle 开关设置，默认开启
- AC3：提交时执行 `openclaw config set gateway.*` 类命令写入配置
- AC4：所有字段有内联说明文字，告知推荐值

---

### US-01-05：可视化 LLM Provider 配置

**作为** AI 开发者，
**我希望** 通过卡片选择内置 Provider 并填写 API Key，或通过表单自定义第三方 Provider，
**以便** 无需了解底层 JSON schema 即可激活大模型服务。

**优先级**：P0
**验收标准（AC）**：
- AC1：内置 Provider（OpenAI、Anthropic、Groq、Moonshot）以卡片形式展示，单击选中
- AC2：选中 Provider 后显示 API Key 输入框，Key 使用系统 Keyring 加密存储
- AC3：自定义 Provider 表单包含 Provider ID、Base URL、Model ID 等必填字段，支持模板快填（Ollama、vLLM 等）
- AC4：必填字段未填时点击"下一步"弹出错误 Toast 并高亮对应输入框
- AC5：Marketplace Tab 提供购买额度并一键激活的商业路径

---

### US-01-06：跳过可选配置步骤

**作为** 希望快速上手的用户，
**我希望** 在 Channel、Skills、Hooks 等可选步骤点击"跳过此步"，
**以便** 先用最基础的配置启动，后续再按需补充。

**优先级**：P1
**验收标准（AC）**：
- AC1：step3（Channel）、step4（Skills）、step5（Hooks）页脚显示"跳过此步"按钮
- AC2：跳过后进度条对应步骤变为灰色虚线，并标记为"-"
- AC3：step6（Review）汇总显示已跳过的步骤列表，提示可在设置中补充
- AC4：step1（Gateway）和 step2（Provider）不显示跳过选项

---

### US-01-07：配置完成后重新运行向导

**作为** 需要重新配置的用户，
**我希望** 在完成页或主界面触发重新运行配置向导，
**以便** 修改已有配置或重新走一遍初始化流程。

**优先级**：P1
**验收标准（AC）**：
- AC1：完成页显示"重新运行配置向导"链接按钮
- AC2：配置向导 Header 右上角有"✕"退出按钮，退出时显示 Toast 提示已保留已完成步骤
- AC3：重新运行时清空跳过状态记录，从 step1 重新开始

---

### US-01-08：稍后配置，先进入主界面

**作为** 想先探索界面的用户，
**我希望** 在安装成功页选择"稍后再说"跳过配置向导，
**以便** 先进入主界面感受应用，配置等之后有空再做。

**优先级**：P1
**验收标准（AC）**：
- AC1：安装成功页提供"稍后再说，先去主界面"按钮
- AC2：点击后 Toast 提示"已保存进度，可在设置中继续配置"
- AC3：跳转到 Bot 管理主界面
