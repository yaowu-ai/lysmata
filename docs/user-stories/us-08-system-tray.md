# US-08：系统托盘 & 快捷入口

**模块**：System Tray
**对应 PRD**：`docs/prd-v2.md §2.1 系统托盘与快捷`

> `US-08` 属于 Phase 2 的桌面伴侣体验增强。v1 先解决“装得上、配得起、选模板、得到第一个可用助手并开始使用”，托盘与快捷入口不进入首版主发布路径。

---

### US-08-01：最小化到系统托盘

**作为** 需要长期运行 Agent 的用户，
**我希望** 关闭主窗口时应用最小化到系统托盘而非退出，
**以便** Gateway 和 Bot 连接保持运行，随时可以唤回。

**优先级**：P1
**验收标准（AC）**：

- AC1：点击窗口关闭按钮时显示"最小化到托盘 / 退出"选项
- AC2：运行期间托盘图标持续显示
- AC3：双击托盘图标或点击"显示"唤起主窗口

---

### US-08-02：托盘图标反映全局状态

**作为** 用户，
**我希望** 通过托盘图标颜色或角标了解当前所有 Bot 的总体状态，
**以便** 不需要打开主窗口即可感知异常。

**优先级**：P1
**验收标准（AC）**：

- AC1：所有 Bot 正常连接时图标正常显示
- AC2：有 Bot 断线或报错时图标显示警告标记（红点或黄点）
- AC3：Hover 托盘图标时显示 Tooltip，摘要各 Bot 状态

---

### US-08-03：托盘右键菜单快捷操作

**作为** 用户，
**我希望** 右键托盘图标出现快捷菜单，
**以便** 不打开主窗口即可执行常用操作。

**优先级**：P1
**验收标准（AC）**：

- AC1：菜单包含：显示主窗口 / 新建对话 / 重启 Gateway / 退出
- AC2：菜单中列出所有 Bot 的连接状态，点击可跳转至对应私聊
- AC3：点击"退出"前确认是否同时停止 Gateway

---

### US-08-04：全局快捷键唤起应用

**作为** 用户，
**我希望** 通过全局快捷键（如 Cmd+Shift+Space）在任意应用中唤起 lysmata 窗口，
**以便** 快速发送 AI 指令而不需要切换任务。

**优先级**：P2
**验收标准（AC）**：

- AC1：快捷键在系统设置中可自定义
- AC2：唤醒时若窗口已在后台则前置，若最小化则恢复
- AC3：快捷键冲突时系统提示并引导修改

---

### US-08-05：Gateway 异常时桌面通知

**作为** 用户，
**我希望** Gateway 断线或 Bot 报错时收到桌面通知，
**以便** 及时处理问题，避免任务静默失败。

**优先级**：P1
**验收标准（AC）**：

- AC1：Gateway 停止时推送桌面通知，附带"重启"快捷操作按钮
- AC2：Bot 连接异常持续超过 30 秒时推送通知
- AC3：通知可在系统设置中关闭
