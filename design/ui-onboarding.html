本地使用保持 127.0.0.1；局域网共享可设为 0.0.0.0

确认配置

确认以下变更，应用并重启 Gateway。

+ "gateway": {
+ "port": 18789,
+ "auth": "none",
+ "autostart": true
+ },
+ "providers": {
+ "primary": "openai",
+ "openai": { "model": "gpt-4o" }
+ },
+ "channels": {
+ "lysmata-desktop": { "enabled": true }
+ }
